# com2c MVP Scaffold

PHP + MySQL scaffold for `com2c.me`

## Included
- Front controller routing
- Basic MVC-ish structure
- Dashboard / cards / inquiries / credits / sponsors pages
- OpenAI service stub
- Credit / sponsor / inquiry summary service stubs
- SQL schema starter

## Deploy notes
- Point your web root to `/public`
- Update `app/config/*.php`
- Import `sql/schema.sql`
- Ensure PHP cURL and PDO MySQL are enabled
